## 三、V3优化版交易数据

### 3.1 新增字段

V3在V2基础上优化了策略，字段相同，但卖出信号更丰富：

| 卖出信号 | 说明 |
|----------|------|
| `持仓到期` | 持有满5天正常卖出 |
| `移动止损(-X%)` | 从最高价回撤X%触发止损 |
| `排名下降(换仓)` | 预测得分掉出前15%换仓 |
| `期末清仓` | 回测期末强制清仓 |

## 四、数据使用说明

```python
import pandas as pd

# 读取V2交易明细
v2_trades = pd.read_csv('docs/backtest_v2_trades.csv')

# 读取V3交易明细
v3_trades = pd.read_csv('docs/backtest_v3_optimized.csv')

    v3_summary = json.load(f)
```

### 4.2 数据分析示例

```python
# 计算月度收益
trades['date'] = pd.to_datetime(trades['trade_date'])
trades['month'] = trades['date'].dt.to_period('M')
monthly_returns = trades.groupby('month')['return'].sum()

# 分析某只股票
stock_data = trades[trades['symbol'] == '600193.SH']
print(f"交易次数: {len(stock_data)}")
print(f"累计收益: {stock_data['return'].sum()*100:.2f}%")

# 分析卖出原因
reason_stats = trades.groupby('sell_signal').agg({
    'return': ['count', 'mean']
})
```
